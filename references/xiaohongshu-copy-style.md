# Xiaohongshu Copy Style

Use this reference when generating Xiaohongshu copy for Juhuo AI event promotion. In v1, QQ Channel, QQ group, and QQ campus wall posts reuse the Xiaohongshu copy and materials by default.

## Positioning

The Xiaohongshu post should be short, friendly, searchable, and easy to repost. It should feel like a campus activity note from a real student community, preferably like a senior student inviting juniors and peers to come hang out, learn, and meet people, not a formal announcement.

## Recommended Structure

```markdown
Title / first line with hook

1-3 short lines explaining who should care.

Core event explanation:
"不是答辩，也没有评委打分..."

Highlights:
✅ ...
✅ ...
✅ ...

适合谁来:
📌 ...
📌 ...
📌 ...

Event info:
⏰ ...
📍 ...
📝 ...

Closing line.

Hashtags.
```

## Length

Default target:

- 250-500 Chinese characters.
- 5-8 bullet lines.
- No long paragraphs.

If the user asks for "更短版", target 180-300 Chinese characters.

## Voice

Prefer:

- Direct invitation.
- Low-pressure wording.
- Slightly lively but not over-cute.
- Senior-student / campus community invitation energy.
- Warm phrases such as "来现场坐坐", "先来听听", "不用准备得很正式", "没想清楚也没关系".
- Concrete verbs: 来看、来聊、来组队、来旁听、来认识同频的人.

Avoid:

- Long策划案 language.
- Too many abstract nouns.
- Overly official tone.
- Activity notice tone that only lists facts.
- Excessive emoji density.
- Selling-like claims.

## Hook Patterns

Good first lines:

- `这周日，来现场看看 AI 怎么从工具箱变成真实项目。`
- `正在做大创 / 创业项目？来聊聊 AI 能怎么接进去。`
- `没有项目也可以来，旁听、组队、认识同频的人都欢迎。`
- `想试试 AI 创业，但还没想好怎么开始？`
- `这周日，学长学姐喊你来现场坐坐。`

## Required Info

Include when available:

- Event identity.
- Who should come.
- What happens on site.
- Time.
- Location or location uncertainty.
- Registration method or QR placeholder.
- CTA.

If a fact is tentative, preserve uncertainty:

- `📍 地点：以报名群通知为准`
- `📝 报名：扫码填写表单 / [此处放报名二维码]`

## Hashtags

Use 8-12 tags. Mix brand, topic, campus, and action tags.

Default tag pool:

```text
#炬火AI #炬火ai #AI #西南大学 #大创 #创业 #组队 #项目落地 #AI工具 #大学生活 #校园活动
```

Do not use tags that imply unconfirmed official sponsorship or guaranteed opportunities.

## QQ Reuse

Use the Xiaohongshu copy and materials as the default QQ宣发 version for:

- QQ Channel.
- QQ group notices.
- QQ campus wall submissions.

Optional light adaptations:

- For QQ groups, remove hashtags if they feel noisy.
- For QQ campus wall, keep public-safe facts only and avoid internal details.
- For QQ Channel, keep hashtags or convert them into topic labels.

Do not create a separate QQ copy unless the user explicitly asks.
